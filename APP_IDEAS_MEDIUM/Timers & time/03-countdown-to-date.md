# Countdown to date (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a multi-event countdown app. Main screen: list of upcoming events sorted by date, each showing event name, date, and days/hours remaining with a progress bar. Add event sheet: name, date picker, icon picker (SF Symbols), color picker. Past events move to Archive tab. Detail screen: full countdown (days, hours, minutes, seconds updating live). Persist with Codable + UserDefaults.
```
