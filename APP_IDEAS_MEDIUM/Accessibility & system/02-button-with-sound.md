# Button with sound (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a soundboard app. Main screen: grid of 8 colored buttons, each playing a different system sound. Labels showing sound name. Volume slider. Favorites row: long-press button to add to favorites (shown at top). Settings: toggle vibration with sound, choose between 3 sound sets. Recently played list. Rapid-fire mode toggle (allow overlapping sounds). Persist favorites and settings with UserDefaults.
```
