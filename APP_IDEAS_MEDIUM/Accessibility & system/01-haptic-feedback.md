# Haptic feedback (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a haptic playground. Main screen: grid of buttons, each triggering a different haptic type (light, medium, heavy, success, warning, error, selection change). Each button labeled with the haptic name. Toggle to enable/disable haptics. Pattern button: plays a sequence of haptics. Settings: intensity slider (maps to different generators). Log of triggered haptics with timestamps. Persist preference with UserDefaults.
```
