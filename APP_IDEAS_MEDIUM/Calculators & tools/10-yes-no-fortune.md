# Yes/No fortune (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a magic 8-ball fortune app. Main screen: 3D-style ball graphic with answer reveal animation. Ask a question (text field), shake or tap to get answer from expanded set (15+ responses across Positive, Neutral, Negative). History tab: list of past questions and answers with timestamps. Favorites: save interesting answers. Stats: most common response type. Custom answers editor. Persist with Codable + UserDefaults.
```
