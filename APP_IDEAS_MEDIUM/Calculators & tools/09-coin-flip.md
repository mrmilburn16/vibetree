# Coin flip (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a coin flip app with streaks. Main screen: large coin graphic with flip animation (3D rotation). Flip button and shake-to-flip gesture. Results: show Heads or Tails with running stats (total flips, heads count, tails count, percentage, current streak). History tab: list of last 50 flips with timestamps. Settings: choose coin style (4 options), toggle sound. Persist stats and history with Codable + UserDefaults.
```
