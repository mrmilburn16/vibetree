# Speed dial (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a speed dial with favorites and recents. Main screen: grid of large contact buttons (3×3) with avatar initials and names. Tap shows action sheet: Call, Message, Email (all show alerts in MVP). Edit mode: rearrange contacts, change grid size (3×3 or 4×4). Add contact sheet: name, phone, email, and color picker for avatar. Recents tab: list of tapped contacts with timestamps. Favorites section pinned at top. Persist with Codable + UserDefaults.
```
