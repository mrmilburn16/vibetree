# Simple counter (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a multi-counter app. Main screen: list of named counters, each showing current value. Tap a counter to open detail screen with large number display, +/- buttons, step size picker (1, 5, 10), and reset button with confirmation. Add counter sheet with name and starting value. Swipe to delete. Persist all counters with Codable + UserDefaults.
```
