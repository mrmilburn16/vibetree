# Notes app (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a notes app with folders and search. Main screen: list of folders (Personal, Work, Ideas). Tap folder to see notes list with title and preview text. Note editor: title field and multi-line text editor with character count. Search bar searches across all folders. Swipe to delete or move to different folder. Sort by date modified. Persist with Codable + UserDefaults.
```
