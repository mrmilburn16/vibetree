# Packing list (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a packing list app with templates. Main screen: list of trips (name, date, progress). Tap trip for sectioned checklist (Clothes, Toiletries, Electronics, Documents). Add item to section. Check/uncheck items. Progress bar showing packed percentage. New trip sheet with name, date, and template picker (Weekend, Business, Beach). Persist with Codable + UserDefaults.
```
