# Segmented content (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a segmented content app with 3 sections. Main screen: segmented control (Feed, Popular, Saved). Feed: chronological list of cards. Popular: same items sorted by likes count. Saved: only items marked as saved. Each card has title, preview text, like button with count, and save button. Tap card for detail screen. Animated segment transitions. Persist saved items with Codable + UserDefaults.
```
