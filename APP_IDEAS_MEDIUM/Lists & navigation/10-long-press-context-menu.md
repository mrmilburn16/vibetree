# Long-press context menu (Medium)

**Prompt (copy-paste into Vibetree Pro)**

```
Build a file manager with context menus. Main screen: list of files/folders with icons, names, sizes, and dates. Long-press shows context menu: Rename, Move, Duplicate, Delete, Share. Tap folder to navigate into it (NavigationStack). Breadcrumb path shown in nav title. Add file/folder buttons. Sort by name, size, or date. Swipe to delete. Persist with Codable + UserDefaults.
```
